\# Layout Rules — Zuzana Kalfus Website



These rules define how layouts should be structured.



Typography, colors and spacing are defined in:



docs/design-system.md



Do not redefine those values here.



---



\# Grid System



Use a centered container.



Container width



1280px max



Container padding



24px



Preferred layouts



single column text  

two column sections  

three column card grids



Avoid complex nested layouts.



---



\# Section Structure



Typical section layout



headline  

short description  

content block  

call to action



Keep sections simple.



---



\# Hero Section



Hero must be visually dominant.



Structure



large headline  

short supporting paragraph  

primary CTA  

secondary CTA



Layout



two column grid



Left



headline + text



Right



visual element or card



Minimum vertical spacing



120px



---



\# Card Layout



Cards must be simple.



Structure



title  

short description



Padding



32px



Border radius



8px



Border



subtle neutral color



Avoid heavy shadows.



---



\# CTA Sections



CTA sections should be minimal.



Centered layout



headline  

short text  

button



Large spacing above and below.



---



\# Image Usage



Images should support content.



Preferred placements



About section  

Project previews



Avoid large decorative imagery.



---



\# Navigation



Navigation should remain simple.



Logo on the left.



Menu items on the right.



Maximum



5 navigation links.



Optional CTA button.



---



\# Footer



Minimal footer.



Copyright  

Navigation links  

Contact link



Avoid large complex footers.



---



\# Design Tone



calm  

minimal  

editorial  

professional  

structured
